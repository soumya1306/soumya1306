## 🚀 Featured Project

### [AI Notes Hub](https://github.com/soumya1306/ai-notes-hub) — [Live Demo](https://ai-notes-hub-omega.vercel.app/)

A production-deployed full-stack GenAI app built solo across 20 incremental phases.

- **RAG Q&A** — Ask natural language questions; Gemini answers grounded in your notes via top-5 pgvector retrieval
- **Semantic Search** — pgvector + Gemini embeddings with HNSW cosine index
- **Real-time Collaboration** — WebSocket per-note rooms with typing indicators and role-based access
- **JWT + Google OAuth** — Refresh token rotation, in-memory access tokens (XSS protection)
- **Full pytest suite** — Unit + integration tests with rollback isolation, GitHub Actions CI
